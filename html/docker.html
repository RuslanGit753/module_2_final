Docker

Общие команды

  • docker info - получение информации о Docker-системе.
  • docker version - просмотр версии Docker.
  • docker help - получение справки по командам Docker.

Управление образами

  • docker pull "имя-образа" - загрузка образа из Docker Hub или другого реестра.
  • docker images - просмотр списка локально сохраненных образов.
  • docker rmi "имя-образа" - удаление образа.
  • docker build -t "имя-образа":"тег" "путь" - создание образа из Dockerfile.

Управление контейнерами

  • docker run "имя-образа" - запуск нового контейнера.
  • docker run -d "имя-образа" - запуск контейнера в фоновом режиме (detached mode).
  • docker ps - просмотр запущенных контейнеров.
  • docker ps -a - просмотр всех контейнеров, включая остановленные.
  • docker stop "идентификатор-контейнера" - остановка запущенного контейнера.

Работа с сетью

  • docker network ls - просмотр всех сетей.
  • ocker network create "имя-сети" - создание новой сети.
  • docker network inspect "имя-сети" - просмотр информации о сети.
  • docker network rm "имя-сети" - удаление сети.

Работа с томами (volumes)

  • docker volume ls - просмотр всех томов.
  • docker volume create "имя-тома" - создание нового тома.
  • docker volume inspect "имя-тома" - просмотр информации о томе.
  • docker volume rm "имя-тома" - удаление тома.

Управление Docker Compose

  • docker-compose up - запуск сервисов, определенных в docker-compose.yml
  • docker-compose down - остановка и удаление контейнеров, созданных с помощью docker-compose
  • docker-compose ps - просмотр состояния сервисов.
