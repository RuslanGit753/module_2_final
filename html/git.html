Git

  • git ls-files – вывести на экран список файлов в папке.
  • cd – расшифровывается как "Change directory", что переводится как "сменить папку" или "перейти в другую папку".
  • git rev-parse --show-toplevel – чтобы узнать корневую директорию вашего репозитория.
  • git init – инициализируйте новый репозиторий.
  • git status – проверьте статус репозитория.
  • git add имя_файла – добавьте файлы для отслеживания.
  • git commit -m "Сообщение о коммите" – после добавления файлов вы можете создать коммит.
  • git status – просмотр статуса репозитория.
  • git log – чтобы посмотреть историю коммитов, используйте команду.
  • git clone – клонирование существующего репозитория.
  • git push -u origin master – отправьте ваши изменения в удалённый репозиторий, основная ветка.
